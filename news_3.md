# Pglens: Herramientas de PostgreSQL para agentes de IA

Pglens es un nuevo proyecto Open Source que ofrece 27 herramientas de solo lectura para PostgreSQL diseñadas específicamente para agentes de IA a través del Model Context Protocol (MCP). Esto permite que los agentes de IA puedan consultar y analizar bases de datos de forma segura, sin riesgo de modificar o corromper los datos, facilitando la integración de LLMs con entornos de producción.
