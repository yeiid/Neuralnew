# Cocoa-Way: Compositor Wayland nativo para macOS

La comunidad Open Source ha lanzado Cocoa-Way, un compositor Wayland nativo para macOS. Esta herramienta permite ejecutar aplicaciones gráficas de Linux sin problemas en el entorno de Apple. Esto es una excelente noticia para los desarrolladores que buscan mantener flujos de trabajo de Linux mientras operan dentro del ecosistema de hardware de macOS.
